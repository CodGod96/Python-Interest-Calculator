This is a calculator for finding the total amounts when accounting for different kinds of interest
** this is also available for use on ti-84 calculators with native python availability **

The Main Menu asks the kind of interest you will be dealing with; input the number at the front of your choice 
(ex. 1. simple interest ; type 1 and hit enter)

The next screen will ask you a number of questions depending on which option you picked on the main menu
Most will have the option of solving for the Total Value after a given time or finding the principal before a given time

** the only exception to this is continuous since you will be able to calculate for the APY instead of the pricipal **

After picking which operation you will be performing it will ask you to input all of your values
** APR will always be inputted as the percentage - DO NOT CONVERT TO DECIMAL **
(ex. APR = 5% ; 5 will be the input, not 0.05)

Once your values are entered it will automatically run the calculations and spit out labelled results for you depending on the type of interest
You will then be given the option to return to the main menu or terminate the program

** system, math, and os will be needed from pip to run the software **
